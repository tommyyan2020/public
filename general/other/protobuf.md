

# ProtoBuff使用

# 下载安装

- 官方

```ba
wget https://github.com/protocolbuffers/protobuf/releases/download/v3.20.1/protobuf-all-3.20.1.tar.gz
tar -zxvf protobuf-all-3.20.1.tar.gz
cd protobuf-3.20.1
./configure --prefix=/usr/local/
make
make check
make install
protoc --version
```
